# DentDesk MVP v0.1

**Версия:** MVP v0.1  
**Цель MVP:** Запустить минимально жизнеспособный продукт, который уже можно продавать стоматологическим клиникам в Алматы.

---

## 🎯 Общее видение MVP

DentDesk — это **WhatsApp-бот + мини-CRM** специально для стоматологий.

Пациент пишет в WhatsApp как обычному администратору (свободным текстом), а внутри работает LLM. Всё выглядит так, будто пациент общается с живым человеком по имени **Айгуль**.

Клиника получает:

- Умного бота 24/7
- Удобный веб-кабинет с диалогами и календарем
- Интеграцию с MacDent (и позже с другими CRM)

---

## 🔄 Полный User Flow

### 1. Onboarding клиники (5–10 минут)

1. Клиника заходит на сайт dentdesk.kz  
2. Нажимает **"Подключить бота бесплатно"**  
3. Регистрируется (email + пароль)  
4. Заполняет короткую форму:
   - Название клиники  
   - Город  
   - Количество врачей  
   - Используете MacDent? (Да/Нет)  
   - Если Да → поле для ввода `access_token` MacDent  
5. Система создаёт аккаунт  
6. Показывает QR-код и ссылку на WhatsApp-номер бота  
7. Клиника пишет **"Привет"** в этот номер → бот активируется  

---

### 2. Личный кабинет клиники (Веб)

#### Диалоги

- Список всех чатов  
- Поиск и фильтры  
- История переписки  
- Ручные ответы  
- Метки пациента  

#### Календарь и кресла

- Календарь по дням  
- Kanban-кресла  
- Drag & Drop  
- Конфликты с MacDent  

#### Пациенты

- База пациентов  
- Карточка пациента  
- История  
- Сообщения  
- Заметки  

---

### 3. Бот (Айгуль)

- Понимает свободный текст  
- Предлагает слоты  
- Создаёт запись  
- Подтверждает  

---

### 4. Интеграция с MacDent

- Проверка слотов  
- Создание записи  
- Синхронизация статусов  

---

### 5. Автоматизация

- Напоминания  
- Отзывы  
- Typing indicator  
- Guardrails

- WhatsApp API  
- Adapter pattern
